#### 功能
CSS（层叠样式表）用于设置和布置网页

#### 层叠规则
CSS 规则的顺序很重要；当应用两条同级别的规则到一个元素的时候，写在后面的就是实际使用的规则。

#### 理解层叠

有三个因素需要考虑，根据重要性排序如下，前面的更重要：

1. 重要程度
2. 优先级
3. 资源顺序

#### 继承
CSS 为控制继承提供了五个特殊的通用属性值。每个 css 属性都接收这些值。
1. inherit
2. initial
3. revert
4. revert-layer
5. unset
